
# FIAP Cloud Games – Fase 1
Desenvolvimento de MVP da plataforma Fiap Cloud Games (FCG).
Nessa etapa está presente:
- Cadastro e autenticação de usuário
- Cadastro, consulta e aquisição de jogos

## Propostas da fase 01
- Criar uma API REST em .NET 8+
- Persistência de dados, segurança, qualidade e testes do software
- Uso de boas práticas durante o desenvolvimento (TDD, DDD, etc)

## Arquitetura
- Monolito;
- DDD;
- TDD
- Briefing e alinhamento inicial a partir de Event Storming.

## Persistência
- Entity Framework Core.
- SQL Server

## API
- .NET 9
- Controllers
- Autenticação via JWT Bearer
- OpenAPI Scalar

## Testes
- Testes unitários para toda a lógica de negócio.
